# Stock Market Predictor and Sentiment Analyzer

This project is a Flask-based web application that predicts stock prices using a trained LSTM model and performs financial sentiment analysis using FinBERT Large language model. It fetches live stock data via `yfinance`, analyzes recent news sentiment, and returns combined insights to the user.

---

## Features

- Predicts future stock prices using LSTM
- Analyzes financial news sentiment using FinBERT LLM
- Retrieves the latest news for the selected company
- Fully deployed on Render (https://swe-app.onrender.com)

---
